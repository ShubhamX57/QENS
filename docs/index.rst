QENS
====

Quasi-Elastic Neutron Scattering analysis toolkit.

Overview
--------

QENS is a Python library for Bayesian and classical analysis of
quasi-elastic neutron scattering data.

Features
--------

- Forward-model Bayesian inference
- Translation + rotation dynamics
- MCMC sampling
- Instrument resolution handling
